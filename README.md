# Assignment 5.1: Will the Customer Accept the Coupon?

## Overview
This project explores the effectiveness of Amazon's marketing strategy, whereby coupons are sent to drivers to attract them to businesses in the drivers' local vincinity. 

## Data
The analysis is based on data collected via a survey on Amazon Mechanical Turk (from UCI Machine Learning Repository).
The data set includes attributes about the driver and their journey, as well as details relating the coupon sent to the drivers - see 'coupon.csv' file appended.

## Toolset
Python - with following libraries
* pandas
* numpy
* matplotlib.pyplot
* seaborn
